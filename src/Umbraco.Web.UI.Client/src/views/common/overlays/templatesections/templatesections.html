Render child template
Renders the entire contents of a child template, by inserting a @RenderBody()
Render a named section
Inserts a @RenderSection placeholder to render a named section of a child template. This allows you to insert specific the parts of a child template which is wrapped in a @section{ ... } definition.
Set the name of the section to render in this area of the template
If mandatory, and the child template does not contain a section with this name, an error will be displayed.
Define a named section
Marks a part of your template as a named section by wrapping it in a @section { ... }, which can be inserted in a specific area of this templates master using @RenderSection.
Give the section a name
